# Install Directories

Generated from built-in Botstack manifests.

## GBrain

- `~/.botstack/cache/gbrain` — Create Botstack cache for the GBrain installer helper
- `~/.codex/config.toml` (codex) — Register GBrain notes for Codex users
- `~/.config/opencode/opencode.json` (opencode) — Register GBrain notes for OpenCode users
- `~/.claude.json` (claude) — Register GBrain notes for Claude Code users
- `~/.continue/config.json` (continue) — Register GBrain notes for Continue users
